# Простая нейронная сеть на Python 

Этот проект представляет собой простую полносвязную нейронную сеть (ANN) с возможностью обучения методом обратного распространения ошибки (backpropagation). Сеть построена вручную (без использования библиотек вроде TensorFlow или PyTorch) и предназначена для демонстрации основных принципов работы нейросетей. 

# На примере реализовано обучение логической операции OR с двумя входами и одним выходом. 
# Структура кода 

    NeuroNetwork — основной класс нейросети   
        Поддерживает произвольное количество скрытых слоёв  
        Активационная функция по умолчанию: сигмоида  
        Производная функции активации используется для backpropagation  
        Скорость обучения (learning_rate) настраивается при создании сети
         

    Layer — слой нейронов   
        Может быть входным (Input), скрытым (Hidden) или выходным (Output)  
        Содержит список нейронов и ссылку на предыдущий слой
         

    Neuron — отдельный нейрон   
        Хранит своё значение и список входов (Input)  
        Вычисляет взвешенную сумму входов и применяет активационную функцию  
        Участвует в обратном распространении ошибки
         

    Input — связь между нейронами   
        Хранит вес (weight) и ссылку на предыдущий нейрон
         
     

# Как работает обучение 

    1. Подается входной вектор (например, [0, 1])  
    2. Выполняется прямое распространение (forward pass) — вычисляются значения всех нейронов  
    3. Сравнивается выход сети с ожидаемым значением — вычисляется ошибка  
    4. Выполняется обратное распространение (backpropagation) — корректируются веса по градиенту  
    5. Повторяется N раз (по числу итераций)
